# 使用手册


## 1. 安装Python

在安装Playwright之前，您需要确保已经安装了Python。您可以在[Python官网](https://www.python.org/downloads/)下载并安装最新版本的Python。安装时请确保选中"Add Python to PATH"选项。

## 2. 创建虚拟环境（可选）

为了避免依赖冲突，建议为您的项目创建一个虚拟环境：

```bash
python -m venv env
```

## 3. 安装Playwright

使用pip命令安装Playwright：

```bash
pip install playwright
```

## 4. 安装浏览器依赖

```bash
python -m playwright install
```
